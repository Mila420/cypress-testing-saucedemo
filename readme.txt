1. Clonar el repositorio o descargar los archivos del proyecto.
2. Abrir la terminal y navegar al directorio del proyecto.
3. Ejecutar `npm install` para instalar las dependencias necesarias.
4. Ejecutar `npx cypress open` para abrir Cypress en modo interactivo.
5. Seleccionar la prueba `sauceDemoTest.cy.js` y ejecutarla.
6. Para ejecutar las pruebas en modo headless, usar el comando `npx cypress run`.
